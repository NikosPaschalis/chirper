# Chirper — Learning Laravel

I'm learning Laravel via the "Getting Started with Laravel" bootcamp while building Chirper, a small Twitter-like app. This README summarizes the 13 lessons and what I learned from each.

## Course overview
13 lessons · ~2 hr

## Lesson notes

1. What are we building? — 1:00
   Overview of Chirper and the final app layout.

2. Setting up your Laravel project — 6:00
   Installing PHP/Composer, creating the project, and preparing my dev environment.

3. Your first route — 8:00
   Creating routes and Blade views; built a simple homepage.

4. Deploying your app — 8:00
   Basics of Git and deploying to Laravel Cloud (or another host).

5. What is MVC? — 11:00
   How Model-View-Controller organizes code; created a controller.

6. Working with the database — 13:00
   Using migrations and seeding to define tables and sample data.

7. Our first model — 12:00
   Creating an Eloquent model to work with database records.

8. Showing the feed — 5:00
   Building Blade components and formatting timestamps for the feed UI.

9. Creating and storing Chirps — 13:00
   Forms, validation, and storing user input safely.

10. Edit and delete Chirps — 15:00
    Implemented edit/delete, learned RESTful routing patterns.

11. Basic authentication: Registration — 14:00
    Built registration, hashed passwords, and auto-login for new users.

12. Basic authentication: Login/Logout — 12:00
    Implemented login, logout, middleware, and protected routes.

13. What's Next? — wrap-up
    Ideas for extending Chirper and next learning steps.
